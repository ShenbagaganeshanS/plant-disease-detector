# 🌿 Plant Disease Detection System

An AI-powered web application that detects plant diseases from leaf images and provides solutions for farmers.

---

## 🚀 Features

- 🌱 Image classification using Transfer Learning (MobileNet)
- 📸 Upload leaf image for instant prediction
- 💡 Suggests solutions for detected diseases
- 🎨 Interactive UI with Streamlit
- 🌍 Environmental awareness slogans

---

## 🛠 Tech Stack

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
